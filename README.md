# Ki-67-automatic-quantification-project
breve descrizione dei file

ottimizza_parametri: calcola i parametri ottimali di segmentazione e classificazione che minimizzano la differenza di KI-67 LI rispetto alle patches analizzate su qupath 

Pos_cell_det: contiene l'analisi completa delle patches(segmentazione con stadist e classificazione con GMM, generazione heatmaps, overlays e tabelle riepilogo)

bland_altman_codici: genera il grafico di bland altman per confrontare metodo manuale e automatico

genera_grafici: genera grafici boxplot per il metodo manuale, automatico e genera un istogramma di confronto tra i due metodi suddivisi per fasce cliniche

analisi_correlazione: genera uno scatter plot e esegue l'analisi di correlazione di spearman tra i due metodi
